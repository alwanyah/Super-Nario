# Super-Nario
![Screenshot (305)](https://user-images.githubusercontent.com/95567513/188307511-743b9968-e2fb-4cef-a25d-95da4f223351.png)
simple 2D run-and-jump game

## conrtol
- ´A´ to go left
- ´D´ to go right
- ´W´ or ´SPACE´ to jump

## installation
1. go ´Builds´
2. install ´Super Nario.zip´
3. extract the files
4. start ´Super Nario.exe´
5. have fun :)
